# Contributing

The first contribution standard is clarity. Notes, docs, project pages, and posts should be easy to edit in a normal text editor and easy to review in Git.

If you are contributing to the public site:

- prefer direct Markdown edits over CMS-only workflows
- keep structure obvious through headings and short sections
- treat overview `index.md` files as navigation hubs, not placeholders
- link outward to deeper project-specific homes when that serves readers better

For Aetheria-specific writing, the primary home remains the dedicated Quartz site at `aetheria.gamecult.org`.
