# Publishing Surfaces

`gamecult.org` and `aetheria.gamecult.org` should complement each other rather than compete.

The root site is for:

- studio identity
- project overviews
- cross-project blog posts
- docs about how the public-facing site works

The Aetheria site is for:

- worldbuilding
- Aetheria-specific essays and stories
- documentation that benefits from the Aetheria visual language
- a structure that can keep evolving around the needs of that project

That split keeps the studio legible without flattening Aetheria into just another subsection.
